# Calligraphy Templates

Parametric templates for calligraphy practice with my light box.

## Project Structure

Once I get the template so it's an actual template, I'll have separate
directories for practice and project templates.

```
calligraphy_templates/
├── README.md
├── LICENSE
├── .gitignore
├── template.tex                # Parameterized document generation logic
├── nibs.tex                    # Nib sizes based on physical measurements
├── practice/                   # Generic templates with repeated guidelines
│   ├── foundational-c3.tex
│   ├── foundational-c5.tex
│   ├── gothic-c3.tex
│   └── ...
└── projects/                   # Bespoke templates for specific projects
    ├── certificate-gothic.tex
    ├── certificate-italic.tex
    └── ...
```
